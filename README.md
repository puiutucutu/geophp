# geophp

A repo for commonly used geo functions in php.

## Run Tests

```bash
git clone <...>
composer install;
cd geo-fns;
clear; ./vendor/bin/phpunit --testdox;

clear; ./vendor/bin/phpunit --bootstrap vendor/autoload.php --testdox tests;
```
